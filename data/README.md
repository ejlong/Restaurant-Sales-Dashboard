This project uses a restaurant sales dataset available from Maven Analytics.

Source: https://mavenanalytics.io/data-playground/restaurant-orders

Description: The dataset contains restaurant order-level data, including items, categories, and transaction details used to analyze sales performance and ordering behavior.

The dataset is publicly available and provided by Maven Analytics.

The raw data is not included as separate files in this repository. Data cleaning, transformation, and analysis were performed in the Excel workbook located in the /excel folder.

The Tableau workbook in the /tableau folder was used to create the final dashboard visualization.
